“湘念黔乡”非遗筑梦之旅项目策划

发表于2024-06-06更新于2024-06-09

“湘念黔乡”非遗筑梦之旅项目策划

“湘念黔乡”非遗筑梦团2024-06-062024-06-09


主办单位：“湘念黔乡”非遗筑梦团
活动时间：2024年7月15日 ——7月20日

一、活动主题

  非遗托起振兴梦，文化联结黔湘情。

二、活动目的及意义

  本项目从科技、文化、国家发展角度在当地计划开展相关课程以及活动，助力当地教育事业。在科技角度，本团队将进行前沿知识、先进科技的科普活动类课程，拓宽同学们的知识面并激起同学们的学习热情；在文化角度，本团队将把湖大特色文化带入丹寨县，与当地的同学们进行文化交流与融合；在国家发展角度，本项目充分与国家发展大方向结合，并不只是一味的进行教育科普类活动，而是更侧重于文化方面的传承和发展，聚焦于非物质文化遗产这一国家重点项目的传承与传播。本团队着重关注于丹寨县特色非遗旅游产业发展，助力其进一步实现乡村振兴发展。同时，本团队也同样考虑到了丹寨县本身少数民族居多的特点，将为当地同学以及居民进行普通话普及活动，助力国家普通话普及进程推进，进而助力乡村振兴发展。

三、活动对象

  “湘念黔乡”非遗筑梦团全体师生、贵州省黔东南苗族侗族自治州丹寨县居民。

日程安排

日期 活动安排 任务

7月16日 A组：前往卡拉村进行非遗项目卡拉鸟笼的学习与传承，走访卡拉村村民，进行调研以及普通话普及工作。 调研报告、新闻稿撰写、社交媒体平台内容发布、分发普通话宣传手册

7月17日 A组：进入当地初中进行“湖大文化交流”活动类课程 课程相关内容设计制作、新闻稿撰写、社交媒体平台内容发布
B组：前往排莫村进行非遗项目苗族蜡染的学习与传承，走访排莫村村民，进行调研以及普通话普及工作。 调研报告、新闻稿撰写、社交媒体平台内容发布、分发普通话宣传手册
7月18日 进入当地初中进行“前沿科技科普”活动类课程 课程相关内容设计制作、新闻稿撰写、社交媒体平台内容发布
7月19日 进入当地小学进行“普通话普及”活动类课程 课程相关内容设计制作、新闻稿撰写、社交媒体平台内容发布、分发普通话宣传手册
7月20日 与丹寨相关负责人员开展座谈会，总结汇报整体项目成果 新闻稿撰写、社交媒体平台内容发布

五、活动前期准备

  1. 与当地相关负责人员取得联系，确定好课程活动所在学校以及班级；
  2. 对丹寨县非遗特色旅游业发展进行预调研，主要调研其发展背景、发展路线、发展现状等，设计调研报告相关问题与内容；
  3. 普通话宣传手册内容编写以及设计制作、活动横幅设计制作；
  4. 非遗宣传网站建立、社交媒体平台账号注册。

六、活动流程

非遗特色旅游业发展座谈会

活动前期

  1. 与相关负责人员取得联系，确定好时间地点。
  2. 充分做好相关背景调查，做好座谈会所需要的问答环节准备。

活动中期

  1. 学习环节：团队全体成员倾听学习负责人员为我们介绍当地非遗文化以及非遗特色旅游业发展现状。
  2. 问答环节：将事先准备好的关于当地发展的相关问题清晰明确地向负责人员提问，更加深入地了解当地民情。
  3. 交流环节：与负责人员深入交流，为其介绍本项目的计划安排，并初步提出本团队对于其乡村振兴发展的创新思路。
  4. 合影留念：全体成员合影，宣布实践项目正式开始。

活动后期

  1. 总结座谈会内容，并根据实际情况完善之后各项活动细节。
  2. 撰写新闻稿，并发布到相关社交媒体平台。

走进卡拉村、排莫村相关活动

活动前期

  1. 各组路线安排人员至少提前两天规划好去往各村的路线、交通工具、所需时间以及费用等相关问题，确保活动顺利进行。
  2. 准备好相关调研材料以及需要分发的普通话宣传手册等，以防遗落。
  3. 各组负责人至少提前一天确定好活动详细人员分工，保证活动高效进行。

活动中期

  1. 学习与传承：进入当地非遗文化基地，了解并学习当地非遗传承史及发展史、当地特色非遗的内涵以及制作工艺等。
  2. 民生与发展调研：走访当地居民，调研民生民情，着重调研当地居民对于当地特色非遗旅游业发展的看法与感受，深入了解当地居民对于现有发展方式以及现今生活的满意程度，总结民众真实心声。
  3. 普通话普及：为当地居民分发普通话宣传手册，并为其介绍普通话普及的意义等相关知识。

活动后期

  1. 整理总结活动调研结果，统计分析调研数据，并根据调研结果分析后续发展中所需要注意到的细节以及需要解决的问题。
  2. 撰写新闻稿，并发布到相关社交媒体平台。
  3. 在非遗宣传网站以及账号上推送相关内容，宣传丹寨县非遗文化以及其特色旅游业。

课程类活动

活动前期

  1. 再次联系相关负责人员确定好课程活动时间、地点等。
  2. 至少提前一天做好相关课程准备，如制作上课所需PPT、视频、课程文稿等。

活动中期

  1. 主题内容讲解：根据各个课程主题，讲解相关知识，预计时间占每节课程的1/2左右。
  2. 互动环节：根据各个课程的主题，设计进行诸如问答、游戏、小剧场等互动活动，提高同学们的积极性，预计时间占每节课程的1/2左右。
  3. 合影留念：课程结束时，与上课班级老师、同学们进行合影留念。

普通话普及课程需发放普通话宣传学习手册。

活动后期

  1. 撰写新闻稿，并发布到相关社交媒体平台。
  2. 每节课程过后总结梳理上课过程中所遇到的问题等，对后续课程内容做出相关调整，以确保活动课程顺利进行。

实践活动总结暨结果报告座谈会

活动前期

  1. 回顾整理整个实践活动过程，做出总结与反思。
  2. 准备好调研报告以及整理出的民生民情反馈。
  3. 与相关负责人员取得联系，确定好时间地点。

活动中期

  1. 总结与反思：根据整理结果，对于整个实践项目过程做出成果总结与问题反思。
  2. 调研结果反馈：根据调研结果，将调研内容、调研数据等反馈给相关负责人员。
  3. 交流与讨论：根据实践过程，提出我们团队的感受以及对于丹寨乡村振兴发展的创新想法及思路，为丹寨更进一步实现乡村振兴事业贡献力量。
  4. 合影留念：团队全体成员与丹寨相关负责人员合影留念。

活动后期

  1. 撰写新闻稿，并发布到相关社交媒体平台。
  2. 整理好各项书面材料，完善实践项目收尾工作。

七、注意事项

  1. 确保好保险购买成功，一切活动在确保成员们人身安全的前提下进行；
  2. 所有活动进行过程中相应成员要做好文字、照片、视频等形式的记录；
  3. 集体车票计划只买去程，大家可以在黄平机场或者凯里南站出发回家，不再统一回长沙；
  4. 所有集体消费开发票，因为目前还不确定互联网+项目能否报销以及报销金额，所以开发票以备所需。
